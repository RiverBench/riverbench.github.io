    Dataset: citypulse-traffic (development version)

    This is a processed subset of real traffic data published by the CityPulse EU FP7 project. The used dataset is 'Road Traffic Data', dataset-4. The dataset was processed to fix prefixes without a trailing '#' or '/'. The dataset consists of subsequent chronologically ordered substreams of observations from each sensor. See also the paper for more details.

    The dataset is published in a few size variants, each containing a specific number of stream elements. For each size, there are three distribution types available: flat (just an N-Triples/N-Quads file), streaming (a .tar.gz archive with Turtle/TriG files, one file per stream element), and Jelly (a native binary format for streaming RDF).
