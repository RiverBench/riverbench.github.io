    Dataset: nanopubs (development version)

    Nanopublications are small units of publishable information, used for scientific results and more. This dataset is based on a subset of a dump of all available nanopublications as of April 5, 2018. Only the first 5M of freely-licensed nanopubs were included. Each nanopub consists of several RDF graphs and thus is an RDF dataset. The included data is primarily from the biomedical domain. More information: paper, website.

    Info

    Download this metadata in RDF: Turtle, N-Triples, RDF/XML, Jelly

    Download links

    The dataset is published in a few size variants, each containing a specific number of stream elements. For each size, there are three distribution types available: flat (an N-Triples/N-Quads file in the RDF Message Log format), streaming (a .tar.gz archive with Turtle/TriG files, one file per stream element), and Jelly (a native binary format for streaming RDF). See the documentation for more details.
